git status - текущая состояние проекта git diff - показывает различия в текущем и сохраненном состоянии git diff --staged показывает проиндексированные изменения git add <имя фала> - сохранение текущий изменений. .(Точка) - сохранить все изменения в файле git commit -m "текст" - добавление описание к сохраненному состоянию git checkout <имя файла> - откатить изменения git checkout <хеш коммита> <имя файла>- откатить изменения коммита git log - выводить историю изменений
